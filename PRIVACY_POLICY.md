# Privacy Policy for Daily Code Reminder

**Last Updated: January 26, 2026**

## Data Collection

This extension collects **NO personal data**. All information is stored locally:

- Task completion status
- Settings (reminder frequency, quiet hours)
- Streak statistics

## What We Don't Do

- ❌ No data sent to external servers
- ❌ No browsing history tracked
- ❌ No personal information collected
- ❌ No analytics or tracking

## Permissions

| Permission | Purpose |
|------------|---------|
| `alarms` | Schedule reminders |
| `storage` | Save settings locally |

## Data Deletion

Uninstall the extension to delete all stored data.

## Contact

Open an issue on GitHub for questions.
